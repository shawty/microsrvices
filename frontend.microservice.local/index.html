Home This is the standard homepage

This page is in a standard HTML file called index.html, located in the root of the main website folder. It's not subject to any routing rules, backend code or anything else along those lines.

It's pure HTML5 markup, with the bootstrap 3 UI framework, Jquery and Knockout JS added to allow clientside data binding and other fun things. The menu accross the top will be set up to use various routes which will appear just as regular leaf nodes, but which infact will actually be handled by owin/katana based service applications running behind the scenes.

The leaf nodes that activate these routes, will be controlled by Application Request Routing directly in IIS7, which is easily installed using the various web platform tools and configuration wizards.
